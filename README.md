# AlfonzM's dotfiles

⚡️ Dotfiles for when my laptop gets stolen, or I get a new computer — hopefully it's the latter.

## Overview

Contains:

- zsh config
- brew installs (and cask apps)
- iTerm2 config
- aliases
- functions

## Installation

```
$ git clone https://github.com/AlfonzM/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ sh install.sh
```
